Classe RemoteConnect

Classe responsável por realizar conexões e operações com uma API remota, incluindo autenticação e gerenciamento de dados de funcionários e construções.

Construtor

__init__(config_path="config.ini")

Lê as configurações do arquivo INI e realiza autenticação imediata com a API.

Métodos

authenticate() → Optional[dict]

Autentica o usuário com email e senha fornecidos na configuração. Retorna um dicionário com os dados de autenticação (incluindo JWT) ou None em caso de falha.

_post(endpoint: str, data: dict) → str

Método interno que realiza um POST autenticado para o endpoint fornecido com os dados informados.

authenticate_user(email: str, password: str) → str

Solicita a criação de uma conta de usuário na API.

register_return(mv) → str

Registra um retorno de movimentação facial na obra. Espera um objeto com os atributos: branchCompanyCustomerConstruct, employeeCode, readding, recordType, nsr.

request_face_download(company: Company, construction_code: str) → str

Solicita o download de dados faciais de funcionários para uma construção.

request_face_update(company: Company, employee) → str

Solicita a atualização dos dados faciais de um funcionário.

request_face_registration_active(company: Company) → str

Solicita a ativação de registros faciais de funcionários de uma empresa.

request_face_remove_active_face(uuid: str) → str

Solicita a remoção de um rosto ativo identificado por UUID.

update_employee(employee_array: List[str]) → str

Atualiza os dados de múltiplos funcionários por lote. Recebe uma lista de strings com os identificadores únicos dos funcionários.

Dependências

  • requests: biblioteca para requisições HTTP
  • configparser.ConfigParser: para leitura de arquivos INI
  • Company: classe representando uma empresa (presumivelmente definida no módulo core.Company)
